About

This application was developed for the purpose of classifying text (messages) for disaster relief. The application was completed as part of a project for the Udacity Data Scientist Nanodegree program. The datasets used to train the supervised learning model were provided by Figure Eight.

The project was completed by Daniel Grewal. If you're interested in the source code behind the application, hit the GitHub Repository link at the top of the page.
